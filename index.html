Lista de Exercícios 3 - Desenvolvimento Web

Esse site foi desenvolvido exclusivamente para praticar os conceitos em HTML e CSS adquiridos nas aulas do curso Dev. Full Stack Jr da +praTi

Missão

Minha missão ao desenvolver este projeto foi aplicar na prática os fundamentos essenciais do desenvolvimento web, com foco em estruturação semântica de páginas HTML, estilização com CSS e criação de componentes interativos. Através dos exercícios, busquei consolidar conhecimentos fundamentais que servirão como base sólida para minha atuação futura como desenvolvedora front-end.

Tecnologias aplicadas

Durante este projeto, utilizei HTML5 para estruturar as páginas de forma semântica, CSS3 para aplicar estilos visuais, incluindo conceitos de Box Model, seletores, Grid e Flexbox. Também experimentei media queries para responsividade e implementei meu primeiro menu interativo adaptado para dispositivos móveis. Cada etapa foi pensada para unir teoria à prática de maneira progressiva.

Visão

A visão deste projeto é ser um marco inicial no meu portfólio como desenvolvedora, refletindo minha evolução e comprometimento com o aprendizado contínuo. Almejo transformar estudos em soluções reais, com código limpo, acessível e responsivo, seguindo boas práticas e princípios modernos do front-end.
